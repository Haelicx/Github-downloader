# 🍴 Step 1 — Fork Repository

Click the fork button at the top right:

```txt
https://github.com/Haelicx/Github-downloader/fork
```

Or manually:

```bash
GitHub → Fork → Create Fork
```

---

# ⚙️ Step 2 — Enable GitHub Actions

After forking:

```bash
Repository → Actions Tab → Enable Workflows
```

GitHub may ask:

```txt
"I understand my workflows, go ahead and enable them"
```

Click it.

---

# 📥 Step 3 — Download Files

Go to:

```bash
Actions → Download Workflow → Run workflow
```

Example URLs:

```txt
https://example.com/movie.mp4
https://example.com/archive.zip
https://example.com/video.mkv
https://example.com/image.png
```

Then:

```diff
+ Choose normal mode
or
+ Choose zip mode
```

Finally:

```bash
Click → Run Workflow
```

---

# 🌐 Step 4 — Capture Websites

Open:

```bash
Actions → Browse Workflow → Run workflow
```

Enter website URL:

```txt
https://example.com
```

The workflow automatically:

```diff
+ Downloads HTML
+ Captures screenshot
+ Extracts images/videos
+ Saves everything into pages/
+ Generates indexes
```

---

# 📂 Step 5 — Organize Files

Run:

```bash
Actions → Sort Workflow → Run workflow
```

Automatically organizes:

```txt
movie.z01
movie.z02
movie.z03
```

Into:

```txt
downloads/movie/
```

---

# 🗑 Step 6 — Clean Downloads

Run:

```bash
Actions → Clean Workflow → Run workflow
```

This will:

```diff
- Delete download files
+ Recreate clean folders
+ Generate README.md
```
